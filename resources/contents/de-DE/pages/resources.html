GitHub Desktop

Du hast dies wahrscheinlich installiert, bevor du Git-it gestartet hast, aber falls nicht, hier ist der Link.

Die GitHub Desktop App lässt dich vieles von dem machen, was du im Terminal kannst (aber nicht alles) und ist manchmal ein bisschen einfacher zu lesen. Persönlich verwende ich es, um alles zu betrachten, was nicht klein ist diff ich habe. Deine Änderungen sind wirklich schön auf diese Weise zu sehen.

  • GitHub Desktop (OSX, Windows) Kostenlos von GitHub
  • GitHub Desktop Anleitungen von GitHub
