Über Git-it

Git-it lehrt die wesentlichen, alltäglichen Git-Befehle, die Entwickler jeden Tag verwenden, zusammen mit dem GitHub Flow, der Art, wie Entwickler auf GitHub zusammenarbeiten.

Einige Challenges benötigen eine Internetverbindung, daher musst du online sein, um Git-it abzuschließen. Aber du kannst die App immer öffnen und die Challenges und Ressourcen offline ansehen.

Echte Entwicklungsumgebung

Git-it verwendet das echte Terminal, Git und GitHub (anstatt sie zu emulieren). Wenn du fertig bist, wirst du echte Repositories in deinem GitHub-Konto und grüne Quadrate in deinem Beitragsdiagramm haben!

Der Code, der Git-it gemacht hat

Diese App ist mit HTML, CSS und JS gebaut und verwendet die Electron Bibliothek. Der gesamte Code für Git-it ist online in seinem Repository auf GitHub. Git-it ist ein Werk der Liebe von @jlord und freundlichen Mitwirkenden.
