Besser zusammenarbeiten

Das Repository, das du bisher erstellt hast, ist nur auf deinem Computer, was praktisch ist, aber es macht es ziemlich schwer zu teilen und mit anderen daran zu arbeiten. Keine Sorge, dafür ist GitHub da!

GitHub ist eine Website, die es Menschen überall ermöglicht, hochzuladen, woran sie mit Git arbeiten, damit jeder einfach an demselben Projekt zusammenarbeiten kann. GitHub fungiert als zentrales Repository für dich und alle anderen zum Teilen. Du pushst deine Änderungen dorthin und holst Änderungen von anderen herunter.

Ein GitHub-Konto erstellen

Besuche github.com und melde dich für ein kostenloses Konto an. High five, willkommen!

GitHub-Benutzername zu Git hinzufügen

Füge deinen GitHub-Benutzernamen zu deiner Git-Konfiguration hinzu. Wir machen das nur für Git-it; es macht es einfacher, die kommenden Challenges zu verifizieren. Speichere ihn genau so, wie du ihn auf GitHub erstellt hast und großschreiben, wo großgeschrieben. Hinweis: Du musst die "<" und ">" nicht eingeben.

Füge deinen GitHub-Benutzernamen zu deiner Git-Konfiguration hinzu:

git config --global user.username <UserName>

Du kannst nochmal überprüfen, was du in deiner Git-Konfiguration gesetzt hast, indem du tippst:

git config --global user.username
{{{ verify_button }}}

Stelle sicher, dass du "user.username" oben tippst und nicht "user.name", was den Namen überschreiben würde, den du in der ersten Challenge gesetzt hast und dich ohne username-Eigenschaft zurücklassen würde! Falls du feststellst, dass du das gemacht hast, ist das okay, wiederhole einfach den Schritt in der ersten Challenge, um deinen Namen hinzuzufügen und vervollständige dann den Schritt oben.

GitHub & Git config Benutzernamen stimmen nicht überein

Ein häufiger Fehler ist, dass dein GitHub-Benutzername nicht der Groß-/Kleinschreibung des Benutzernamens entspricht, den du mit git config gesetzt hast. Zum Beispiel ist 'JLord' nicht dasselbe wie 'jlord'

Um deinen mit Git gesetzten Benutzernamen zu ändern, führe einfach denselben Befehl aus, den du vorher gemacht hast, aber mit der korrekten Groß-/Kleinschreibung:

git config --global user.username <UserName>

Wenn du deine Updates gemacht hast, verifiziere nochmal!
