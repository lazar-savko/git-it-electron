Git

Git ist kostenlose Open-Source-Software (der Quellcode ist öffentlich), geschrieben von Linus Torvalds, der auch den Linux-Betriebssystem-Kernel geschrieben hat.

Git ist ein Programm zur Nachverfolgung von Änderungen über die Zeit, bekannt in der Programmierung als Versionskontrolle. Wenn du bereits eine Änderungsverfolgung in einem Textbearbeitungsprogramm verwendet hast, dann kennst du das Konzept bereits!

Git installieren

Wir empfehlen, Git auf deinem Computer zu installieren, indem du die GitHub Desktop App herunterlädst. Wir werden die Desktop-App in Git-it nicht verwenden (wir lernen das Terminal!), aber sie enthält Git und ist der einfachste Weg, Git auf allen Plattformen gleich zu installieren.

  • Windows: Verwende die Git Shell als dein Terminal.
  • Mac: Öffne GitHub Desktop und wähle in den Einstellungen die Installation der Befehlszeilentools aus. Verwende die Terminal App als dein Terminal.

Hast du bereits Git oder bist du dir nicht sicher? Tippe git --version in dein Terminal und wenn es eine Versionsnummer höher als 1.7.10 zurückgibt, bist du bereit! Für weitere Informationen besuche die Git Website.

Git Software

Die GitHub Desktop App kann viele Dinge mit Git machen, aber nicht alles, deshalb ist das Lernen des Terminals wichtig. Aber sobald du das beherrschst, wirst du froh sein, die Desktop-App zu haben, da sie die Informationen deines Projekts visueller organisiert, wie die GitHub Website.

Git allein ist nicht wie andere Programme auf deinem Computer. Du wirst wahrscheinlich kein Icon auf deinem Desktop sehen, aber es wird immer für dich verfügbar sein und du kannst es jederzeit über dein Terminal oder Git Desktop Anwendungen aufrufen.

Git konfigurieren

Sobald GitHub Desktop (und damit Git) installiert ist, öffne dein Terminal. Du kannst überprüfen, dass Git wirklich da ist, indem du tippst:

git --version

Dies gibt die Version von Git auf deinem Computer zurück und sieht etwa so aus:

git version 1.9.1

Als nächstes konfiguriere Git, damit es weiß, deine Arbeit dir zuzuordnen:

Setze deinen Namen:

git config --global user.name "Dein Name"

Setze jetzt deine E-Mail:

git config --global user.email "deineemail@beispiel.com"

Dollar-Zeichen in der Code-Dokumentation
