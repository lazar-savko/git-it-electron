Halte deine Datei auf dem neuesten Stand, indem du Änderungen von Kollaborateuren einholst.

Pullen von einem Remote

Wenn du an etwas mit jemand anderem arbeitest, musst du mit den neuesten Änderungen auf dem Laufenden bleiben. Du wirst also pullen wollen, um alle Änderungen einzuholen, die möglicherweise zum zentralen GitHub-Repository gepusht wurden.

Was hat Reporobot gemacht?

Sieh, ob Reporobot Änderungen an deinem Branch gemacht hat, indem du alle Änderungen vom Remote namens 'origin' auf GitHub einholst:

git pull <REMOTENAME> <BRANCHNAME>

Wenn sich nichts geändert hat, wird es dir 'Already up-to-date' sagen. Wenn es Änderungen gibt, wird Git diese Änderungen in deine lokale Version mergen.

Hat Reporobot Änderungen gemacht? Git sagt dir, wo Änderungen gemacht wurden. Du kannst diese Datei öffnen und Reporobots Updates sehen. Überraschung, Reporobot ist ein Künstler!
